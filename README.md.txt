# Vehicle Insurance Fraud Detection

## Overview

This project uses Machine Learning to predict whether a vehicle insurance claim is **Fraudulent** or **Genuine** based on customer, vehicle, and claim details.

## Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest

## Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Encode Categorical Data
5. Train-Test Split
6. Train Machine Learning Models
7. Evaluate Models
8. Predict Fraudulent Claims

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

## Files

* `Vehicle_Insurance_Fraud_Detection.ipynb` – Jupyter/Google Colab notebook
* `vehicle_fraud.csv` – Dataset
* `requirements.txt` – Required Python libraries

## How to Run

1. Download or clone the repository.
2. Install the required libraries:

   ```
   pip install -r requirements.txt
   ```
3. Open the notebook in Google Colab or Jupyter Notebook.
4. Run all the cells.
